# 🎮 Videojuego: Figuras Geométricas

## 🧩 Descripción:
Este videojuego consiste en controlar una figura geométrica que evoluciona a medida que interactúa con otras figuras en pantalla. El jugador comienza como un triángulo con un color aleatorio (azul, rojo o amarillo) y debe avanzar a través de distintos niveles cumpliendo objetivos específicos.

## 🎯 Objetivo del juego:
El objetivo principal es hacer crecer tu figura aumentando su número de lados mediante colisiones correctas, mientras evitas colisiones incorrectas que la debiliten.

## 🕹️ Mecanicas del juego:
Inicio: Comienzas como un triángulo (3 lados).
Tu figura tiene un color aleatorio: 🔵 Azul, 🔴 Rojo, 🟡 Amarillo.

- ✅ Si colisionas con una figura del mismo color:
Ganas +1 lado.
Tu figura evoluciona: Triángulo → Cuadrado → Pentágono → ...
Aumenta tu puntaje.
- ❌ Si colisionas con una figura de distinto color:
Pierdes -1 lado.
Retrocedes en evolución: Cuadrado → Triángulo.
Disminuye tu puntaje.
- ⏱️ Cada nivel tiene un tiempo límite. Si el tiempo llega a 0, pierdes automáticamente.
